# Politique de Confidentialité - Bot Seraphim

**Dernière mise à jour : 23 novembre 2025**

## 1. Données Collectées et Stockées

Le bot collecte et stocke sur Replit les données suivantes :
- **Identifiants Discord** (obligatoire pour identifier les utilisateurs)
- **Noms d'utilisateur** (pour affichage uniquement)
- **Avertissements et sanctions** (warns, bans, timeouts)
- **Niveaux XP et statistiques d'expérience**
- **Durée en salons vocaux**
- **Nombre de messages envoyés**
- **Données de tickets** (titre, description, date)
- **Badges et récompenses**
- **Configurations serveur** (couleurs, messages personnalisés, filtres)

## 2. Utilisation des Données

Les données sont utilisées pour :
- Fonctionner correctement
- Modérer les serveurs
- Gérer les systèmes de niveaux et expérience
- Personnaliser le serveur selon vos préférences

## 3. Suppression de Données sur Demande

Si un utilisateur demande la suppression de ses données personnelles, seules les données suivantes seront supprimées :
- **Durée en salons vocaux**
- **Nombre de messages envoyés**
- **Données de tickets** (titre, description, historique)

Les autres données (XP, badges, avertissements, etc.) sont considérées comme des données de serveur et ne seront pas supprimées.
